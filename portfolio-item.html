My first offical dive into website design! This was the first project I completed, learning HTML, CSS and BootStrap to get the design & functions of this website.

Being introduced to coding with HTML was a very pleasant experience as it is intuitive and the results are very quickly visible.

Using BootStrap became easier the more i work on this project. It was definetely a learning experience as what to use BootStrap for, how i could then customise the features to my liking. It took time to realise what I could do with BootStrap, it's limitations in customisation and whether I was expending more energy trying to fight with its pre-programmed styles to get what I wanted or if I should just use CSS to get exactly the look I wanted.
