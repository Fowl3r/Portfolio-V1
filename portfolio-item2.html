My Text Based Adventure Game

Escape To The Observatory!

A text based adventure game, was an expansion on the JavaScript knowledge. Learning a lot about classes and functions. Using Object Orientated programming to design the game and how it plays.

Using the Object Orientated Programming approach to this game was very useful. It let me plan the design of the game first and how I wanted to make it look. Then I was able to start adding in objects, functions and behaviours to make the game play how I wanted.

This approach was very useful, it made everything clearer in my mind to have my Objectives planned out infront of me. Especially as i went a little bit overboard with the mapping of the rooms. Having 16 in total and them all succesfully linked up for navigation was a little excessive in retrospect, however I am happy how this has turned out
